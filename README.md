## Introduction
It provides an OpenNMT trainer that creates a bi-lingual translation model.

## Requirements
 * `Python 3`
 * `PIP 3`

## Usage
### Run
```bash
# Install dependencies and run the app.
pip install -r requirements.txt
python ./main.py <ARGUMENTS>
```
